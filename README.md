# 12v Battery Pack 3s4p 18650
3d printed portable battery pack. Could be used in a 3x4 or 4x3 arrangement. Modular stackable sections for different outputs.

## Parts
* 1x [PD charging bi-directional board](https://www.aliexpress.com/item/1005009726309124.html)
* 3x [PD65W Fast Charging modules](https://www.aliexpress.com/item/1005007016771773.html)
